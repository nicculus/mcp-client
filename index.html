MCP over the wire, simplified

TypeScript client library and CLI for connecting to a serverless MCP server over Streamable HTTP.

npm install -g @nicculus/mcp-client

Features

SDK + CLI

Use as an importable TypeScript library or a ready-made terminal command.

Streamable HTTP

Built on the MCP SDK's StreamableHTTPClientTransport — no WebSocket setup needed.

Any cloud

Works with MCP servers deployed on AWS, GCP, or Azure via mcp-infra.

Header auth

Pass x-api-key, Authorization, or custom headers via env or flag.

CLI usage

Set your endpoint and key once, then explore any MCP server from the terminal.

# Set credentials (or place in .env)
export MCP_ENDPOINT=https://your-endpoint.example.com/mcp
export MCP_HEADERS="x-api-key:YOUR_KEY"

# List available tools
mcp-client tools list

# Call a tool
mcp-client tools call get_repo_summary --args '{"repo_url":"https://github.com/owner/repo"}'

# Machine-readable output
mcp-client tools list --json

# Per-request headers (repeatable)
mcp-client tools list --header "x-api-key:KEY" --header "X-Custom:foo"

One-off use without a global install:

npx @nicculus/mcp-client tools list

SDK usage

Import MCPClient in any TypeScript or Node.js project. Each method opens a fresh connection, calls the server, and closes — no persistent state to manage.

import { MCPClient } from "@nicculus/mcp-client";

const client = new MCPClient({
  endpoint: "https://your-endpoint.example.com/mcp",
  headers: { "x-api-key": "YOUR_KEY" },
});

// List available tools
const tools = await client.listTools();
console.log(tools.map(t => t.name));

// Call a tool
const result = await client.callTool({
  name: "get_repo_summary",
  arguments: { repo_url: "https://github.com/anthropics/anthropic-sdk-python" },
});
console.log(result.content);

Environment variables

The CLI reads these at startup. Both can be placed in a .env file in the working directory.

MCP_ENDPOINT

Full URL to the MCP server — e.g. https://abc123.execute-api.us-east-1.amazonaws.com/mcp

MCP_HEADERS

Comma-separated key:value pairs — e.g. "x-api-key:s3cr3t,X-Custom:foo"

Requirements

  • Node.js 18+
  • An MCP server endpoint — deploy one with mcp-infra
